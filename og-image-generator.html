HOW TO EXPORT THIS AS og-image.png
1. Open this file in Google Chrome (file://… or via Live Server in VS Code)
2. Make sure browser zoom is exactly 100% — press Ctrl+0 (Windows/Linux) or Cmd+0 (Mac)
3. Open DevTools: F12 or right-click → Inspect
4. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac) to open the Command Palette
5. Type screenshot and select "Capture full size screenshot"
6. Chrome downloads a PNG — rename it to og-image.png
7. Place it at: assets/images/og-image.png in your GitHub repo
8. Commit and push — the OG image will be live at:
